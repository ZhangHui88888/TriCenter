# 企业数据导入字段映射

> Excel 来源：`docs/data/企业服务数据新表_品类已匹配.xlsx`（1214行，399条有效企业）
> 目标：通过系统现有的**调研Excel导入**功能（`SurveyExcelServiceImpl`）批量导入

---

## 现有导入模板 Sheet 结构

系统调研导入模板包含 7 个 Sheet，每 Sheet 通过隐藏的「企业ID」列或「企业名称」关联：

| Sheet | DTO 类 | 说明 |
|-------|--------|------|
| Sheet0 填写说明 | — | 字段填写要求与可选值参考 |
| **Sheet1 企业基本信息** | `SurveyBasicInfoData` | 企业主表字段 |
| **Sheet2 联系人信息** | `SurveyContactData` | `enterprise_contacts` 表 |
| **Sheet3 产品信息** | `SurveyProductData` | `enterprise_products` 表 |
| **Sheet4 外贸信息** | `SurveyTradeData` | 外贸相关字段 |
| **Sheet5 跨境电商信息** | `SurveyCrossBorderData` | 跨境相关字段 |
| **Sheet6 合作与政策信息** | `SurveyCooperationData` | 政策、竞争力、风险等 |
| Sheet7 需求分析 | 转置矩阵 | 每行一条需求，每列一个企业，填是/否 |

## 需预处理的转换规则汇总

| 转换项 | 源格式 | 目标格式 | 复杂度 |
|--------|--------|---------|--------|
| **员工规模** | 数字（500） | 档位文本（"200-500人"） | 中 — 需查 `staff_size` 选项定义区间 |
| **成立时间** | 纯年份（2023）或字符串 | `YYYY-01-01` | 低 |
| **外贸数据** | "2000万" / 1000 | 纯数字 | 低 — 去除"万"等文字（列23，84条有值） |
| **外贸模式** | 贸易代码（0110） | 系统 label 文本 | 中 — 需查 `trade_mode` 选项的 value→label |
| **跨境平台** | "国际站" | "阿里巴巴国际站" | 低 — 简单替换 |
| **跨境占比** | 小数（0.5） | 百分比（"50%"） | 低 |
| **竞争地位** | "中小企业" | "中型企业" | 低 — 简单替换 |
| **产品品类拆行** | 逗号分隔多品类 | 多行产品记录 | 中 |
| **需求匹配** | 逗号分隔需求文本 | 矩阵是/否 | 高 — 需模糊匹配需求名 |
| **国内电商经验** | "是"/"Y" | "是" | 低 |

## 已确认决策

1. ✅ **列23（外贸数据）**对应外贸营业额（`last_year_revenue`）；**列8（年营业额）**= 总营业额，不导入
2. ✅ **员工规模**自动映射为档位
3. ✅ **行业竞争地位**：中小企业→中型企业
4. ✅ **联系情况/希望支持/合作需求**不导入
5. ✅ **产品品类**已与 `product_categories` 对齐（如"电声元器件"→ID 60106）
6. ✅ **外贸模式** 0110 已确认存在，其他代码待确认
7. ✅ **跨境平台**"国际站"→"阿里巴巴国际站"
