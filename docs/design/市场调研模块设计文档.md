# 市场调研模块设计文档

> 版本：v1.0 | 日期：2026-03-12

## 一、模块概述

市场调研模块为跨境电商三中心提供全方位的市场情报支持，集成联网数据查询、AI智能分析、竞品对比和报告生成能力，帮助工作人员快速掌握全球跨境电商市场动态，为企业招商和服务决策提供数据依据。

### 核心能力

| 能力 | 说明 |
|------|------|
| 联网数据查询 | 对接海关总署、Statista、Google Trends 等数据源，实时获取市场数据 |
| AI 智能分析 | 接入 GPT-4o / DeepSeek 等大模型，自动生成市场分析报告 |
| 竞品分析 | 多维度对比竞品数据，雷达图 + SWOT 矩阵可视化 |
| 报告模板 | 预置市场分析模板，支持自定义模板，一键生成专业报告 |

---

## 二、页面结构

侧边栏菜单新增「市场调研」一级菜单，包含 5 个子页面：

```
市场调研（TrendingUpOutlined）

```

## 三、页面详细设计

### 3.1 调研总览（Dashboard）

**路由**：`/market-research/overview`

**功能定位**：全局数据概览，快速了解市场全貌

| 区域 | 组件 | 数据说明 |
|------|------|----------|
| KPI 卡片 ×4 | StatCard | 全球市场规模、跨境电商增速、热门品类数、市场机会指数 |
| 全球市场趋势 | ECharts 折线图 | 北美/欧洲/亚太 三条趋势线，2020-2026 |
| 热门品类分布 | ECharts 环形图 | 电子产品、服装配饰、家居用品等品类占比 |
| 市场竞争力雷达 | ECharts 雷达图 | 市场规模/增长潜力/竞争强度/政策环境/物流基础 |
| 区域增长率 | ECharts 横向柱状图 | 东南亚、中东、拉美等区域增长率排名 |
| AI 洞察摘要 | 文本卡片 | AI 自动生成的 3 条市场洞察要点 |

**数据来源**：
- 静态数据：定期更新的市场报告数据（存储在数据库）
- 动态数据：AI 定时分析生成洞察摘要

---

### 3.2 数据查询（Data Query）

**路由**：`/market-research/data-query`

**功能定位**：联网搜索 + 多源数据聚合查询

#### 核心交互

1. **搜索栏**：关键词输入 + 联网搜索开关
2. **筛选器**：数据来源（海关总署/Statista/阿里研究院/Google Trends）、时间范围、地区、品类
3. **AI 智能推荐**：根据用户搜索历史推荐相关数据
4. **搜索结果列表**：分页表格展示，支持查看/下载/收藏
5. **数据预览面板**：选中数据的摘要、关键指标、迷你图表
6. **实时数据流**：底部滚动展示政策变化、促销数据等实时信息

#### 数据源接入

| 数据源 | 接入方式 | 数据类型 |
|--------|----------|----------|
| 海关总署 | API / 爬虫 | 进出口贸易数据 |
| Statista | API | 行业报告、市场统计 |
| Google Trends | API | 搜索趋势、热度指数 |
| 阿里研究院 | API / RSS | 电商行业分析 |
| 自有数据库 | 直连 | 企业调研数据 |

---

### 3.3 AI 智能分析

**路由**：`/market-research/ai-analysis`

**功能定位**：AI 驱动的深度市场分析

#### 核心交互

1. **分析控制面板**
   - 问题输入框：自然语言描述分析需求
   - 快捷提示词：市场规模预测、竞争格局分析、消费者画像、价格策略、渠道分析、SWOT分析
   - 模型选择：GPT-4o / DeepSeek

2. **分析结果区域**
   - 分析报告（左侧 55%）：结构化的 AI 分析报告，含数据表格和关键发现
   - 市场规模预测（右上）：组合图表，历史数据 + AI预测 + 置信区间
   - 消费者画像（右下）：年龄分布、客单价、复购率、关键词云

3. **操作按钮**：导出报告、保存为模板、继续深入分析

4. **历史记录**：时间轴展示历史分析，支持查看和复用

#### AI 接入方案

| 模型 | 用途 | 调用方式 |
|------|------|----------|
| GPT-4o | 深度分析、报告生成 | OpenAI API |
| DeepSeek | 数据分析、趋势预测 | DeepSeek API |
| 自定义 Prompt | 模板化分析 | 预置 Prompt + 用户输入 |

---

### 3.4 竞品分析

**路由**：`/market-research/competitive`

**功能定位**：多品牌多维度竞争对比

#### 核心交互

1. **分析配置栏**：品类选择、目标市场、竞品多选（最多5个）、分析维度勾选
2. **综合评分雷达图**：产品力/价格竞争力/品牌影响力/渠道覆盖/营销效果/用户口碑
3. **市场份额趋势**：堆叠面积图，按季度展示份额变化
4. **详细对比表**：价格区间、产品线数量、平台评分、月销量、复购率、品牌搜索指数
5. **价格分布对比**：箱线图展示各品牌价格分布
6. **SWOT 分析矩阵**：2×2 网格，优势/劣势/机会/威胁

#### 数据获取

- 平台公开数据（商品价格、评分、销量）
- 搜索指数（Google Trends / 百度指数）
- AI 辅助分析（自动生成 SWOT）

---

### 3.5 调研报告

**路由**：`/market-research/reports`

**功能定位**：报告管理 + 模板中心

#### Tab 1：我的报告

| 区域 | 说明 |
|------|------|
| 统计卡片 ×4 | 报告总数、本月新增、AI生成、已分享 |
| 报告画廊 | 卡片式布局，缩略图+标题+标签+操作（查看/编辑/分享） |
| 新建报告 | 从模板创建或 AI 生成 |

#### Tab 2：模板中心

| 模板类型 | 预置模板 |
|----------|----------|
| 市场分析 | 市场规模分析、行业趋势报告 |
| 竞品分析 | 竞品对比分析、品牌力评估 |
| 用户研究 | 消费者画像、需求调研 |
| 策略规划 | SWOT分析、进入策略评估 |

每个模板包含：预设图表类型、数据字段映射、报告结构大纲。

## 四、技术方案概要

### 后端新增

| 模块 | 说明 |
|------|------|
| `MarketResearchController` | 调研总览、数据查询 API |
| `AiAnalysisController` | AI 分析请求/结果 API |
| `CompetitiveController` | 竞品分析 API |
| `ResearchReportController` | 报告 CRUD + 模板管理 API |
| `DataSourceService` | 数据源适配层（海关/Statista/Trends） |
| `AiService` | 大模型调用封装（GPT-4o / DeepSeek） |

### 数据库新增表

| 表名 | 用途 |
|------|------|
| `market_data` | 市场数据缓存（来源、品类、地区、指标值） |
| `ai_analysis_records` | AI 分析历史记录 |
| `competitive_profiles` | 竞品档案 |
| `research_reports` | 调研报告 |
| `report_templates` | 报告模板 |
| `data_source_configs` | 数据源配置 |

### 前端新增

| 文件/目录 | 说明 |
|-----------|------|
| `pages/MarketResearch/` | 市场调研模块页面目录 |
| `pages/MarketResearch/Overview.tsx` | 调研总览 |
| `pages/MarketResearch/DataQuery.tsx` | 数据查询 |
| `pages/MarketResearch/AiAnalysis.tsx` | AI 分析 |
| `pages/MarketResearch/Competitive.tsx` | 竞品分析 |
| `pages/MarketResearch/Reports.tsx` | 调研报告 |
| `api/marketResearch.ts` | API 接口封装 |
| `stores/marketResearchStore.ts` | Zustand 状态管理 |
