# 企业详情 - 导出Excel功能

## 功能概述

在企业详情页顶部提供「导出Excel」按钮，点击后将当前企业的所有标签页数据导出为一个多Sheet的Excel文件（`.xlsx`格式）。每个标签页对应一个Sheet，表格样式正式整洁。

## 技术实现

- **前端库**：`exceljs` + `file-saver`
- **工具文件**：`frontend/src/utils/exportEnterpriseExcel.ts`
- **调用位置**：`frontend/src/pages/EnterpriseDetail.original.tsx` 顶部导出按钮

## Excel Sheet 结构

| Sheet名称 | 标签色 | 包含内容 |
|-----------|--------|---------|
| 基本信息 | 紫色 | 企业信息（14个字段）+ 联系人信息（7列） |
| 产品信息 | 绿色 | 产品列表（9列）+ 自主品牌 + 核心技术/专利 |
| 外贸信息 | 橙色 | 外贸开关 + 基本信息 + 业绩分析 + 市场/模式/品类变化 + 原因分析 |
| 跨境电商 | 紫色 | 跨境开关 + 平台列表 + 基本信息 + 目标市场占比 |
| 需求分析 | 青色 | 企业画像维度选择 + 自定义需求 |
| 合作 | 粉色 | 三中心合作状态 + 合作项目/顾虑 + 评估评分 + 标杆可能性 + 补充说明 |
| 竞争力与风险 | 红色 | 调研状态 + 行业竞争地位 + 当前面临风险 |
| 跟进记录 | 深紫 | 跟进记录表格（日期/类型/内容/跟进人/阶段变化） |

## 联系人信息字段（对应数据库 `enterprise_contacts` 表）

| 字段 | 数据库列名 | Excel列名 | 必填 |
|------|-----------|-----------|------|
| 姓名 | name | 姓名 | 是 |
| 电话 | phone | 电话 | 是 |
| 职位 | position | 职位 | 否 |
| 邮箱 | email | 邮箱 | 否 |
| 微信 | wechat | 微信 | 否 |
| 备注 | remark | 备注 | 否 |
| 主要联系人 | is_primary | 是否主要联系人 | 否 |

## 表格样式规范

- **标题行**：主题紫色背景（#667EEA），白色加粗字体，居中对齐
- **数据行**：斑马纹交替（白色/#F8FAFC），左对齐，自动换行
- **分区标题**：左侧彩色边框，浅灰背景，主题色加粗字体
- **Sheet标题**：合并单元格，14号字体，底部紫色粗边框
- **字体**：统一使用微软雅黑

## 导出文件命名

```
{企业名称}_企业信息_{日期}.xlsx
```

示例：`新建企业_1770877342310_企业信息_2026-02-12.xlsx`
