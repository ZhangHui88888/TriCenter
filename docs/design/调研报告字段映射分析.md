# 调研报告字段映射分析

> 分析市场调研报告（基础版）所需的企业信息字段，与企业管理系统数据库字段的对照关系。
> 区分哪些字段已覆盖、哪些字段缺失、以及哪些字段应由 AI 分析生成。

---

## 1. 字段分类说明

| 标识 | 含义 |
|------|------|
| ✅ 已覆盖 | 数据库已有对应字段，可直接填充 |
| ⚠️ 部分覆盖 | 数据库有相关字段但粒度/维度不完全匹配 |
| ❌ 缺失 | 数据库中完全没有对应字段，需要新增 |
| 🤖 AI 生成 | 不应存储在企业数据库中，应由 AI 基于企业信息分析生成 |

## 6. 缺失字段汇总与建议

### 6.1 建议新增到 `enterprises` 表的字段

**已落地**：`has_overseas_distributors`（TINYINT，是否有海外分销商）— 企业详情「外贸信息」Tab，企业列表与调研跨境电商 Sheet 可导入导出；2026-03-22 起已移除 `social_media_accounts`、`exhibition_history`、`overseas_distributors`（文本）、`using_crm`。

| 字段名 | 类型 | 说明 | 建议加在企业详情的位置 |
|--------|------|------|---------------------|
| `established_date` | DATE | 成立日期 | 基本信息 Tab |
| `registered_capital` | VARCHAR(50) | 注册资本（如"500万元"） | 基本信息 Tab |
| `aeo_certification` | VARCHAR(50) | 海关AEO认证等级 | 资质认证区域（新增） |
| `iso_certifications` | VARCHAR(500) | ISO认证情况（如"ISO9001:2015"） | 资质认证区域（新增） |
| `other_certifications` | VARCHAR(500) | 其他资质证书 | 资质认证区域（新增） |

### 6.2 建议新增到 `enterprise_products` 表的字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `export_ratio` | VARCHAR(20) | 出口占比（如"60%"） |
| `profit_margin` | VARCHAR(20) | 利润率（如"15-20%"） |

### 6.3 不建议加入数据库的字段（由 AI 分析生成）

以下字段属于**市场分析结论**，不是企业的固有属性，应在生成报告时由 AI 实时分析：

- 贸易政策与关税、消费水平、电商渗透率、文化偏好（依赖目标市场变化）
- 买家画像、买家痛点、未满足需求（依赖行业动态）
- 行业规模、增长率、发展阶段、趋势（需要实时数据）
- SWOT 分析、竞争对手分析（综合判断）
- 产品质量/差异化/定价评估（主观分析）
- 风险评估、综合评分、行动建议（报告结论）

## 7. 报告字段数据流向总结

```
┌─────────────────────────────────────────────────┐
│                  企业管理系统                      │
│                                                   │
│  基本信息  →  名称/信用代码/成立时间/注册资本/类型   │
│  联系人    →  姓名/职务/电话/邮箱                   │
│  资质认证  →  进出口权/ISO/AEO/行业认证             │
│  产品信息  →  产品线/认证/区域/销量/产能             │
│  外贸信息  →  目标市场/平台/物流/团队/是否有海外分销商 │
│  跨境电商  →  平台/物流/ERP 等                    │
└───────────────┬─────────────────────────────────┘
                │ 企业数据作为输入
                ▼
┌─────────────────────────────────────────────────┐
│              DeepSeek AI 分析引擎                  │
│                                                   │
│  输入：企业的行业+产品+目标市场+资质+规模           │
│  联网搜索：公开市场数据、行业报告、政策法规          │
│  输出：市场分析、买家画像、竞争格局、风险评估等      │
└───────────────┬─────────────────────────────────┘
                │ AI 分析结果
                ▼
┌─────────────────────────────────────────────────┐
│              市场调研报告（基础版）                  │
│                                                   │
│  第一章：企业信息（100% 数据库）                    │
│  第二章：目标市场（数据库目标市场 + AI 市场分析）    │
│  第三章：买家画像（100% AI 生成）                   │
│  第四章：行业竞争（数据库竞争地位 + AI 行业分析）    │
│  第五章：产品分析（数据库产品 + AI 竞争力评估）      │
│  第六章：营销渠道（数据库平台 + AI 渠道分析）        │
│  第七章：风险评估（100% AI 生成）                   │
│  第八章：综合评估（数据库合作需求 + AI 评分建议）    │
└─────────────────────────────────────────────────┘
```
