## 前端控制台诊断日志

- 企业详情页打开时，可在浏览器控制台筛选 `[EnterpriseDetail]` 查看页面级加载阶段日志，包括企业详情、行业树、字典选项、产品分类、需求配置、跟进记录的开始/完成/失败与耗时。
- 所有前端 API 请求会输出 `[HTTP][start]`、`[HTTP][success]`、`[HTTP][error]`，其中包含请求方法、URL、状态码与 `durationMs`，可快速判断是静态页面卡住还是接口本身慢/报错。
- 排查企业详情页白屏或加载久时，优先看 `/enterprises/{id}`、`/options/*`、`/enterprises/{id}/follow-ups` 这些请求的耗时与失败日志。
