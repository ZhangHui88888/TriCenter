# 文档目录说明

> TriCenter — 常州跨境电商三中心企业信息管理系统

---

## 目录结构

```
docs/
├── README.md          # 本文件 — 文档导航索引
├── plan/              # 项目管理：里程碑、模块级任务清单
├── design/            # 需求与设计：需求文档、功能规格、交互流程（按模块拆分）
├── tech/              # 技术文档：架构、API设计、数据库、部署、运维
├── test/              # 测试记录（按模块组织）
└── sql/               # 数据库脚本（schema.sql + init.sql + scripts/）
```
