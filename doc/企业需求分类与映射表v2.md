## 三、维度1：企业类型 → 差异化需求

### 工厂型企业

**画像**：有自有生产线，原做OEM/ODM代工，想转型跨境

**核心优势**：成本控制、产品定制能力、供应链稳定

**核心短板**：品牌意识弱、营销能力缺、终端市场认知不足

| 需求模块 | 对应需求文档编号 | 差异化说明 |
|----------|------------|------------|
| **品牌建设** | 1.1.1 品牌定位与规划/设计、1.7.1 知识产权布局 | 补贴OEM转OBM短板 |
| **运营能力** | 1.8.1 组织架构设计、1.8.2 人才招聘与培养、1.8.3 自建团队/代运营选择 | 营销团队从0到1 |
| **渠道开拓** | 2.1.1 平台开店、2.1.2 独立站建设、2.1.3 线下渠道搭建 | 终端渠道建设 |
| **知识产权** | 1.7.1 知识产权布局、3.4.3 知识产权维护 | 保护自有技术/设计 |

---

### 贸易型企业

**画像**：有货源渠道和供应商资源，无自有生产能力

**核心优势**：选品灵活、供应商资源丰富、市场嗅觉敏锐

**核心短板**：产品差异化难、利润空间受挤压、供应链不可控

| 需求模块 | 对应需求文档编号 | 差异化说明 |
|----------|------------|------------|
| **选品与测品** | 1.4.1 平台测品、双轨选品、1.4.3 品类规划与产品矩阵 | 核心竞争力 |
| **供应商管理** | 1.6.3 采购渠道拓展、3.6.2 供应商评估、3.6.3 合同与结算 | 供应链稳定性 |
| **差异化** | 2.4.1 外包装设计、4.4.2 产品定义 | 突破同质化 |
| **成本与定价** | 3.5.1 多币种定价、3.5.2 毛利/净利核算模型 | 利润空间保障 |

---

### 初创/SOHO

**核心优势**：决策快、试错成本低、灵活性高

**核心短板**：资金有限、资源匮乏、抗风险能力弱

| 需求模块 | 对应需求文档编号 | 差异化说明 |

| **选品策略** | 1.4.1 平台测品、双轨选品 | 快速试错迭代 |
| **学习资源** | 1.8.2 人才招聘与培养、1.5.4 行业协会与展会资源 | 能力快速提升 |

## 八、需求叠加计算示例

> **计算公式**：企业需求清单 = 通用型需求（必选底座 + 按需增强） + 各维度差异化需求

### 示例1
> **企业画像**：工厂型 + B2C平台 + 启动期 + 有品牌
>
> **需求清单** = 通用型需求（必选底座 + 按需增强） + 工厂型差异化 + B2C平台差异化 + 启动期差异化 + 有品牌差异化
>
> **重点模块**：
> - 通用：合规底线、收款与财务基础、物流方案选型、基础客服售后、基础风险管理、市场洞察
> - 工厂型：品牌建设、运营能力、渠道开拓、知识产权
> - B2C平台：开店入驻、Listing优化、广告投放、达人/社媒/直播、防损与退货成本
> - 启动期：开店落地、产品上架、包装与样品、基础推广
> - 有品牌：品牌战略、品牌视觉、知识产权、品牌传播、品牌保护、品牌体验、用户运营

### 示例2
> **企业画像**：贸易型 + B2B平台 + B2C平台 + 增长期 + 无品牌
>
> **需求清单** = 通用型需求（必选底座 + 按需增强） + 贸易型差异化 + B2B+B2C差异化 + 增长期差异化 + 无品牌差异化
>
> **重点模块**：
> - 通用：合规底线、收款与财务基础、物流方案选型、基础客服售后、基础风险管理、市场洞察
> - 贸易型：选品与测品、供应商管理、差异化、成本与定价
> - B2B+B2C：询盘与报价、样品转化、Listing优化、广告投放
> - 增长期：广告放量、转化优化、供应链扩容、团队扩张、多渠道拓展
> - 无品牌：选品为王、成本优先、快速迭代、流量效率、风险分散、合规底线

### 示例3
> **企业画像**：初创/SOHO + 独立站 + 观望期 + 有品牌
>
> **需求清单** = 通用型需求（必选底座 + 按需增强） + 初创差异化 + 独立站差异化 + 观望期差异化 + 有品牌差异化
>
> **重点模块**：
> - 通用：合规底线、收款与财务基础、物流方案选型、基础客服售后、基础风险管理、市场洞察
> - 初创/SOHO：低成本启动、资金规划、选品策略、学习资源
> - 独立站：建站搭建、流量获取、数据隐私、转化优化、私域运营
> - 观望期：成本测算、路径规划、团队规划、风险预判
> - 有品牌：品牌战略、品牌视觉、知识产权

### 建议1：增加「目标市场」维度？

- 日韩
- 中东
- 拉美
- 澳洲

### 建议2：增加「产品类型」维度？
不同品类的合规、物流、运营差异大：
- 消费电子（认证复杂）
- 服装配饰（季节性强、退货率高）
- 家居用品（体积大、物流成本高）
- 美妆个护（FDA/CPNP等特殊合规）
- 食品（严格准入、保质期管理）
- 母婴产品（安全认证严格）
- 汽配（专业性强、长尾市场）

### 建议3：增加「年营收规模」维度？
不同规模企业的资源和需求优先级不同：
- 年营收 < 100万
- 年营收 100-500万
- 年营收 500-2000万
- 年营收 > 2000万
